# Karate Tournament Management System

## Project Description

The Karate Tournament Management System is a full-stack web application designed to manage karate tournaments efficiently. It allows administrators to register participants, schedule matches, manage tournament brackets, and track match results in real time.

This system helps tournament organizers automate tournament workflows and maintain accurate records.

---

## Features

- Participant registration and management
- Match scheduling
- Tournament bracket management
- Real-time results tracking
- Admin dashboard
- Firebase backend integration

## Architecture

The system follows a client-server architecture.

Frontend:
- Built with React and Vite
- Provides user interface for tournament management

Backend:
- Firebase Authentication for secure login
- Firestore Database for storing participants, matches, and results
- Firebase Functions for backend processing

## Installation

Clone the repository:

git clone https://github.com/ranjitha1414/ranjitha1414-karate-tournament-system.git

Navigate to project folder:

cd ranjitha1414-karate-tournament-system

Install dependencies:

npm install

Run the project:

npm run dev
